Hill Climbing - Algoritmo de Optimización

Introducción

El Hill Climbing es un algoritmo de optimización heurística utilizado en problemas donde se busca maximizar o minimizar una función objetivo. Es una técnica de búsqueda basada en iteraciones que mejora progresivamente una solución inicial explorando soluciones cercanas y eligiendo la mejor alternativa en cada paso.

Concepto General

El algoritmo Hill Climbing se inspira en el concepto de escalar una montaña: partimos de una posición inicial y realizamos pequeños cambios en la solución actual con el objetivo de encontrar una mejor alternativa. En cada iteración:

  • Se generan soluciones vecinas.
  • Se compara su desempeño con la solución actual.
  • Se selecciona la mejor solución encontrada.
  • Si no hay mejoras, se detiene el proceso.

Este algoritmo es eficiente para encontrar óptimos locales, aunque puede quedar atrapado en soluciones subóptimas si la función objetivo tiene múltiples máximos o mínimos.

Ventajas y Desventajas

✅ Ventajas:

  • Simple de implementar.
  • Bajo consumo de recursos computacionales.
  • Útil en problemas de búsqueda en grandes espacios de soluciones.

❌ Desventajas:

  • Puede quedarse atrapado en óptimos locales.
  • No garantiza encontrar la mejor solución global.
  • No explora todas las posibilidades, lo que lo hace menos efectivo en funciones complejas.

Ejemplo de Implementación en Python

Aquí tienes un ejemplo práctico de cómo implementar Hill Climbing en Python para optimizar una función matemática:

import numpy as np

def objective_function(x):
    """Función objetivo: maximizar f(x) = -x**2 + 4*x"""
    return -x**2 + 4*x  

def hill_climbing(start_x, step_size=0.1, max_iterations=100):
    x = start_x
    for _ in range(max_iterations):
        neighbors = [x + step_size, x - step_size]
        best_neighbor = max(neighbors, key=objective_function)

        if objective_function(best_neighbor) > objective_function(x):
            x = best_neighbor
        else:
            break  # Óptimo local alcanzado
            
    return x, objective_function(x)

best_x, best_value = hill_climbing(start_x=0)
print(f"Máximo encontrado en x={best_x} con valor {best_value}")

Aplicaciones de Hill Climbing

  • Optimización de hiperparámetros en modelos de Machine Learning.
  • Optimización de rutas en problemas logísticos.
  • Ajuste de configuraciones en sistemas de inteligencia artificial.
  • Resolución de problemas de planificación en sistemas operativos y automatización.
